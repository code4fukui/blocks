# ブロックス

ブロックをドラッグアンドドロップして自由にレイアウトできるシンプルなゲームです。ユーザーはURLを共有することで、ブロックの配置状態を他者と共有できます。

## デモ
[ブロックス デモ](https://code4fukui.github.io/blocks/)

## 機能
- ドラッグアンドドロップでブロックを自由に配置できる
- ブロックの配置状態をURLに保存できる

## 使い方
ブラウザでWebページを開くと、ドラッグアンドドロップでブロックを移動できます。
ブロックの配置状態はURLに保存されるので、URLをコピーして共有できます。
