# blocks

JavaScriptベースのブロック組み立てゲームです。

## デモ
[blocks DEMO](https://code4fukui.github.io/blocks/)

## 機能
- ブロックをドラッグ＆ドロップしてさまざまな形を作成
- ブロックがグリッドにスナップして整列
- ブロックの配置を保存および読み込み可能

## 使い方
[デモページ](https://code4fukui.github.io/blocks/)にアクセスすることで、ブラウザ上で直接プレイできます。ブロックをドラッグ＆ドロップして、さまざまなデザインを作成できます。ブロックの配置はURLを使用して保存および読み込みが可能です。
