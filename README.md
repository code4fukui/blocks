# blocks

A JavaScript-based block building game.

## Demo
[blocks DEMO](https://code4fukui.github.io/blocks/)

## Features
- Drag and drop blocks to create various shapes
- Blocks snap to a grid for alignment
- Ability to save and load block configurations

## Usage
The game can be played directly in the browser by visiting the [demo page](https://code4fukui.github.io/blocks/). Blocks can be dragged and dropped to create different designs.
